gui module¶

This module creates a GUI for touchless mouse to operate. It initilizes the hand tracker and displays the output and video feed on screen.

Routine Names

Inputs

Outputs

Exceptions

Assumptions:

  • __init__ is called before any other access routine.

State variables

gui.window¶

(tkinter.TK()) Creates a toplevel widget of Tk GUI Toolkit which is the main window of an application

window.title¶

(str) Name of the GUI given by the user

gui.video_source¶

(int) Number corresponds to the video source. Initialized at 0 for webcame

gui.hand_tracker¶

(HandTracker) Access the device’s camera and performs handtracking form the camera’s video feed

gui.translator¶

(Translator) Translates the coordinates of hand landmarks into an OS action depending on the change in actions/coordinates.

gui.canvas¶

(tkinter.canvas) graphics for GUI

gui.delay¶

the delay(ms) that requests tkinter to refresh by calling the update() function

gui.update¶

Class method to update the frame from the video feed from hand_tracker.

window.mainloop¶

Tells Python to run the Tkinter event loop. This method listens for events, such as button |clicks or keypresses, and blocks any code that comes after it from running until the window it’s called on is closed.

State invariants

  • video_source >= 0 ∧ video_capture

  • canvas.width >= 0 ∧ canvas.height >= 0

  • ``delay >= 0 ``
