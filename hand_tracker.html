State invariants

  • width > 0 ∧ height > 0

  • success ^ |current_hand_landmarks.Right.landmark| = 20 ∧ (|past_hand_landmarks.Right.landmark| = 20 ∨ |past_hand_landmarks.Right.landmark| = 0)

  • success ^ |current_hand_landmarks.Left.landmark| = 20 ∧ (|past_hand_landmarks.Left.landmark| = 20 ∨ |past_hand_landmarks.Left.landmark| = 0)

class hand_tracker.HandTracker(video_source=0)¶

Bases: object

This is a class that captures live video feed using openCV and tracks handmovment using mediapipe. This live video feed is turned into frames with an overlay onto the tracked hand(s) and returns a singular frame when called.

Parameters

video_source (int) – A reference to which video source should be used (0 is the default webcam of the machine)

Raises

ValueError –

If the video capture is unable to be opened by the given video source, a value error will be raised on that given video source|

get_image_data()¶

Takes a frame from the video feed and tracks and overlays the hand(s) of the user.

Returns

A Tuple containing:

  • success (bool): If frame was captured successfully

  • image (image): The current frame with the hand overlay

  • current_hand_landmarks (Dict): The current left/right landmakrs of each hand

  • past_hand_landmarks (Dict): The past left/right landmakrs of each hand

Return type

tuple
